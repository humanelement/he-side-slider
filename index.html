Arguments

wrap (required, string-selector). jQuery selector to designate which HTML element is "wrap" parent for all child elements related to the slider.

slider (required, string-selector). jQuery selector to designate which HTML element is the "slider" element for this particular slider.
The "slider" element is the moving element. The "slider" should be a parent element above the "view" element(s).

view (required, string-selector). jQuery selector to designate which HTML element(s) are "view" for this particular slider.
A "view" element holds one or more items. A slider only shows one "view" at a time. When sliding next or previous, a different "view" slides into visibility.
"View" element(s) should be children of the "slider" element.

next_btn (optional, string-selector). jQuery selector to designate which HTML element is the "next" button.

prev_btn (optional, string-selector). jQuery selector to designate which HTML element is the "previous" button.
